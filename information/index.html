Information about WebGL

WebGL is a JavaScript API for rendering interactive 2D and 3D graphics within the web browser. WebGL elements can be mixed with other HTML elements and composited with other parts of the page or page background. WebGL programs consist of control code written in JavaScript and shader code that is written in a C-like programming language, which is OpenGL ES Shading Language (GLSL).
